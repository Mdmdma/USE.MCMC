# Euclidean metric on specified columns

`euclidianMetric` calculates the euclidean metric of specified columns
of two points given as dataframes

## Usage

``` r
euclidianMetric(point.a = NULL, point.b = NULL, dim = "")
```

## Arguments

- point.a:

  First point

- point.b:

  Second point

- dim:

  vector containing the names of the columns that should be included in
  the computation

## Value

euclidean metric of the two points
